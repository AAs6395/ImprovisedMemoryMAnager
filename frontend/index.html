Algorithm Info
01
Input Validation

Name uniqueness and size bounds are checked.

02
Buddy Rounding

When enabled, size is rounded up to the next power of 2 (e.g. 100 → 128 KB).

03
Best-Fit Search

Scans all free blocks, picks the smallest one that fits to minimise wasted space.

04
Block Split

If the chosen block is larger than needed, the remainder is returned as a new free block.

05
Auto-compact Fallback

If no block fits, compaction is attempted automatically before failing.
